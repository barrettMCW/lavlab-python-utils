Welcome to LavLab Python Util's documentation!
==============================================

**LavLab Python Util** is a Python library for accessing and processing cloud data.
It pulls data from OMERO and converts it to general purpose formats like numpy arrays.

Check out the :doc:`installation` section for a guide on installing lavlab-python-utils.

Check out the :doc:`getting_started` section for usage information. 

.. note::

   This project is under active development.
